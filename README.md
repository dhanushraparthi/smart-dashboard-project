\# Smart Data Analytics Dashboard



This project is a Python-based dashboard that automatically generates data visualizations from uploaded datasets.



\## Features

\- Upload CSV, Excel, or JSON datasets

\- Automatic column detection

\- Interactive filters

\- KPI metrics

\- Category analysis

\- Correlation heatmap

\- Trend analysis

\- Data insights



\## Technologies Used

\- Python

\- Streamlit

\- Pandas

\- NumPy

\- Plotly
